Notion des Fonctions en JavaScript

En JavaScript, une fonction est un bloc de code réutilisable qui permet d’effectuer une tâche spécifique. On peut la déclarer avec le mot-clé function ou avec une expression de fonction (fonction anonyme ou fléchée). Les fonctions peuvent recevoir des paramètres et retourner une valeur avec return. Elles facilitent la structuration, la réutilisation et la maintenance du code. 

    function addition(a, b) {
        return a + b;
    }
    console.log(addition(2, 3)); // Affiche 5

Notion des Objets en JavaScript

En JavaScript, un objet est une collection de propriétés, où chaque propriété est une paire clé-valeur. Les objets permettent de regrouper des données et des fonctions liées. On peut créer un objet en utilisant des accolades {} ou le constructeur new Object(). Les objets sont fondamentaux pour la programmation orientée objet en JavaScript. 

    const personne = {
        nom: "John",
        age: 30,
        saluer: function() {
            console.log("Bonjour, je m'appelle " + this.nom);
        }
    };
    personne.saluer(); // Affiche "Bonjour, je m'appelle John"

Notion des Tableaux en JavaScript

En JavaScript, un tableau est une structure de données qui permet de stocker une collection d'éléments. Les tableaux sont dynamiques et peuvent contenir des éléments de différents types. On peut créer un tableau en utilisant des crochets [] ou le constructeur new Array(). Les tableaux offrent de nombreuses méthodes pour manipuler les données, comme push(), pop(), shift(), unshift(), etc. 

    const fruits = ["pomme", "banane", "orange"];
    fruits.push("kiwi"); // Ajoute "kiwi" à la fin du tableau
    console.log(fruits); // Affiche ["pomme", "banane", "orange", "kiwi"]

Notion des Promesses en JavaScript

En JavaScript, une promesse est un objet qui représente une opération asynchrone. Elle peut être dans l'un des trois états : en attente (pending), résolue (fulfilled) ou rejetée (rejected). Les promesses permettent de gérer les opérations asynchrones de manière plus lisible et structurée, en utilisant les méthodes then() et catch(). 

    const promesse = new Promise((resolve, reject) => {
        setTimeout(() => {
            resolve("Opération réussie");
        }, 2000);
    });
    promesse
        .then(result => {   
            console.log(result); // Affiche "Opération réussie" après 2 secondes
        })
        .catch(error => {
            console.error("Erreur :", error);
        });

Notion des Événements en JavaScript

En JavaScript, les événements sont des actions ou des occurrences qui se produisent dans le navigateur, comme un clic de souris, une touche enfoncée, ou le chargement d'une page. On peut écouter et réagir à ces événements en utilisant la méthode addEventListener(). Les événements permettent d'interagir avec l'utilisateur et de rendre les pages web dynamiques. 

    const bouton = document.getElementById("monBouton");
    bouton.addEventListener("click", () => {
        alert("Bouton cliqué !");
    });
